# eslint-config-mercari

## Install
```
$ npm install --save-dev eslint-config-mercari
```

## Usage
Add `"extends": "mercari"` to your .eslintrc.
This includes base + react + flow rules.

### base
`"extends": "mercari/base"`

### base + react
`"extends": "mercari/react"`

### base + flow
`"extends": "mercari/flow"`
